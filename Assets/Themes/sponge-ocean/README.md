# sponge-ocean

`sponge-ocean` is an original undersea sponge-style theme generated by WPF
geometry and brushes at runtime.

- No official SpongeBob character images, logos, trademarks, or source art are used.
- The yellow porous shape, bubbles, wave border, coral stroke, and star-style
  ornament are procedural overlay decorations.
- Future bitmap or JSON theme assets for this theme can be placed in this folder.
