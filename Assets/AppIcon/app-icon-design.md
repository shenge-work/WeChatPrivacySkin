# App Icon Design

The v2 icon is a deterministic, code-rendered mark:

- Shape: rounded chat bubble as the outer silhouette, shield in the center.
- Meaning: WeChat conversations protected by a local privacy layer.
- Palette: soft WeChat green, mint highlight, cream background, subtle teal shadow.
- States: the same base icon supports tray state badging later with green, gray, and orange dots.
- Delivery: `app.ico` contains 16, 24, 32, 48, 64, 128, and 256 px frames.

The icon is intentionally vector-like and reproducible rather than AI-generated, so rebuilds can preserve the same brand mark.
